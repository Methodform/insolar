# Инсоляр — версия на React + Radix (Vite)

Переписанный интерфейс сервиса на **React** и **Radix Themes** с 3D-вьюпортом на **Three.js**.
Это рабочая основа архитектуры (движок + UI на Radix); продвинутые функции портируются
из ванильной версии (`../index.html`) поверх этой базы.

## Что уже есть

- Vite + React 18 + `@radix-ui/themes` + Three.js.
- Ядро астрономии (`src/engine/astronomy.js`) — позиция солнца, восход/закат/полдень, разбор контура.
- 3D-вьюпорт (`src/three/Viewport.jsx`) — земля, участок по контуру, забор, солнце и тени
  в реальном времени, орбитальная камера.
- UI на Radix (`src/App.jsx`) — тема (светлая/тёмная), вкладки, карточки-панели, поля,
  Select, Slider-таймбар, статистика по солнцу.

## Запуск

Требуется Node.js 18+.

```bash
cd react-app
npm install       # установка зависимостей
npm run dev       # локальный дев-сервер (http://localhost:5173)
npm run build     # сборка статики в dist/
npm run preview   # предпросмотр сборки
```

## Публикация

`npm run build` создаёт папку `dist/` — это статический сайт, который можно
разместить на GitHub Pages, Netlify, Vercel и т.п. (в vite.config задан `base: './'`,
поэтому работает и из подпапки).
