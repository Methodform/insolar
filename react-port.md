# Инсоляр — React + Radix порт (`react-app/`, деплой в `/app/`)

_Обновлено: 2026-07-18_

## Готово (паритет с ванильной версией)
- Астрономия, положение/движение солнца, тени (Viewport.jsx)
- Здания: типовые пресеты, двускатные крыши
- Гизмо перемещения/поворота/масштаба + drag в 3D
- Инсоляция центроида, длина тени, полярный путь солнца (SunPath)
- Pro-режим, годовая тепловая карта (HeatMap), отчёт, сохранение/загрузка
- **Ручное рисование плана (PlanEditor.jsx)** — новое:
  - прямоугольник (marquee), параллельный сторонам участка (rectFromDrag)
  - контур по точкам с углами 90° (двойной клик / «Замкнуть»)
  - перетаскивание зданий, поворот (R), удаление (Del)
  - линии отступов 1 / 3 м внутрь участка (offsetInward), сетка, метки площади
  - подложка-чертёж (загрузка картинки, ширина в метрах, прозрачность)
  - кнопка «✏️ Рисовать» на левой панели

## Осталось для паритета
- Инсоляция по окнам
- Рекомендации по зонированию
- Модалка памятки нормативных отступов
- Ограничение размещения по отступам (в 3D)

## Движок (astronomy.js) — гео-хелперы
polyAreaSigned, polyArea, nearestOnSeg, clampToPoly, offsetInward, plotBasis, rectFromDrag — проверены `node --check`.

## Деплой
Изменения в `react-app/` требуют commit+push → CI (deploy.yml) пересобирает `/app/`.
