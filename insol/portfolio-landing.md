**Дата старта:** 2026-07-17
**Владелец:** Marat (mabdrashitov@ilotcos.com)

## Бриф
- Тип: одностраничный конверсионный лендинг-портфолио
- Профессия: UX / продуктовый дизайнер
- Главная цель (конверсия): посетитель связывается (форма / кнопка «Связаться»)
- Стилистика: Apple / Samara — светлый фон, крупная типографика, много воздуха, скролл-анимации

## Структура страницы
1. Nav (sticky, минимальный)
2. Hero — крупный заголовок, подзаголовок, primary CTA
3. Selected Work — сетка кейсов
4. About — коротко о себе + навыки
5. Process — как я работаю (3–4 шага, в духе Samara «how it works»)
6. Testimonials / логотипы (соц. доказательство)
7. Contact — финальный CTA + форма
8. Footer

## Технические решения
- Один файл `portfolio-landing.html`, без внешних зависимостей
- Системные шрифты (SF-подобные), CSS-переменные
- IntersectionObserver для fade/slide скролл-анимаций
- Полностью адаптивный (mobile-first)
