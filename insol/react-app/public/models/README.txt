Модель дерева для 3D-сцены (опционально)
=========================================

Положите сюда файл модели с именем:  tree.glb

После сборки он окажется по пути /app/models/tree.glb и автоматически
подхватится сценой вместо процедурной ёлки-конуса. Если файла нет — сцена
тихо использует конусные ёлки (ничего не ломается).

Где взять бесплатно (лицензия CC0 — можно в коммерции, без указания автора):
- Kenney "Nature Kit"        — https://kenney.nl/assets/nature-kit  (низкополигональные .glb)
- Quaternius "Nature Kit"    — https://quaternius.com/               (CC0, .glb/.gltf)
- Poly Haven                 — https://polyhaven.com/models          (CC0)

Как подготовить:
1. Скачайте набор, выберите хвойное дерево/ель.
2. Экспортируйте/переименуйте в  tree.glb  (формат .glb, желательно < 1–2 МБ).
3. Положите файл в эту папку:  react-app/public/models/tree.glb
4. Закоммитьте и запушьте — при сборке файл попадёт в /app/models/tree.glb.

Требования: одиночная модель дерева, начало координат внизу (ствол на земле).
Масштаб не важен — сцена сама подгонит высоту под заданную высоту дерева.
