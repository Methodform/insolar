# Инсоляр — моделирование солнца и инсоляции участка

Веб-сервис для моделирования положения и движения солнца, теней и инсоляции земельного
участка по координатам. Помогает спланировать размещение дома, бани, огорода и посадок
ещё до начала стройки и подготовить нормативный расчёт инсоляции по СанПиН.

**Демо (после включения GitHub Pages):** `https://<username>.github.io/<repo>/` — приложение,
`.../landing.html` — лендинг.

## Возможности

- Положение и движение солнца в реальном времени по координатам и дате (алгоритм Meeus/SunCalc).
- 3D-сцена с реальными тенями, диаграмма пути солнца, восход/закат/полдень, длина дня.
- Ввод участка по координатам угловых точек; красные линии отступов 1/3 м.
- Здания: типовые сооружения, рисование прямоугольником/контуром, двускатная крыша,
  перетаскивание и 3D-гизмо (перемещение по X/Y/Z, поворот, изменение длины/ширины).
- Ограничение размещения в границах участка с нормативным отступом по типу объекта
  (СП 53.13330 / СП 42.13330).
- Забор по периметру с учётом в тенях и инсоляции.
- Подложка: чертёж (JPG/PNG) или спутниковый снимок по координатам (Яндекс/Google Static API,
  нужен ваш API-ключ).
- **Pro-режим:** инсоляция по сетке точек и по окнам, годовая тепловая карта,
  3D-аналитика поверхностей (тепловизор), рекомендации по зонированию, нормативный отчёт
  (PDF), сохранение/открытие проекта.
- Памятка по нормативным отступам; калькулятор экономии/риска; светлая/тёмная тема.

## Запуск

Это статический сайт без сборки. Достаточно открыть `index.html` в браузере
(нужен интернет для загрузки Three.js с CDN).

Локальный сервер (по желанию):

```bash
python3 -m http.server 8080
# затем открыть http://localhost:8080/
```

### GitHub Pages

1. Settings → Pages → Source: `Deploy from a branch`, ветка `main`, папка `/root`.
2. Через ~минуту сайт будет доступен по адресу из раздела Pages.
