# Инсоляр — React + Radix порт (`react-app/`, деплой в `/app/`)

_Обновлено: 2026-07-18 (итерация 3)_

## Готово (паритет с ванильной версией)
- Астрономия, положение/движение солнца, тени (Viewport.jsx)
- Здания: типовые пресеты, двускатные крыши
- Гизмо перемещения/поворота/масштаба + drag в 3D
- Инсоляция центроида, длина тени, полярный путь солнца (SunPath)
- Pro-режим, годовая тепловая карта (HeatMap), отчёт, сохранение/загрузка
- **Ручное рисование плана (PlanEditor.jsx)** — новое:
  - прямоугольник (marquee), параллельный сторонам участка (rectFromDrag)
  - контур по точкам с углами 90° (двойной клик / «Замкнуть»)
  - перетаскивание зданий, поворот (R), удаление (Del)
  - линии отступов 1 / 3 м внутрь участка (offsetInward), сетка, метки площади
  - подложка-чертёж (загрузка картинки, ширина в метрах, прозрачность)
  - кнопка «✏️ Рисовать» на левой панели

## Готово (добавлено 2-й итерацией)
- **Инсоляция по окнам** (facadeWindows + windowInsolation + windowsReport в движке; Pro-диалог)
- **Памятка нормативных отступов** — модалка (свободно)
- **Псевдо-облака + компас-надписи С/Ю/В/З** в 3D-сцене (приближаются/отдаляются при зуме)
- **Рекомендации по зонированию** — ZoneMap.jsx (юг огород / центр газон / север посадки), Pro-диалог
- **Ограничение отступов в 3D** — перетаскивание/поворот/масштаб держат объект в участке (дом ≥3 м, прочее ≥1 м; offsetInward + pointInPoly)

## Готово (итерация 3)
- **Иконки @radix-ui/react-icons** вместо эмодзи во всём интерфейсе App
- **3D-аналитика поверхностей «тепловизор»** — плавный градиент по земле/фасадам/крышам/забору (raycast против мешей сцены), выбор месяцев, учёт рассеянного света, легенда-шкала (Viewport.jsx: thermalColor/gridSurface/roofSlopes3D + эффект аналитики)
