## 1. Обзор

«Инсоляр» — веб-сервис для моделирования положения и движения солнца, теней и инсоляции участка по координатам. Помогает спланировать размещение дома, бани, огорода и посадок ещё до стройки и подготовить нормативный расчёт инсоляции по СанПиН.

**Ценностное предложение (DoD):** «Узнай за 15 минут, куда поставить дом, баню и сад — пока не поехал бетон» (для частника); «Отчёт по инсоляции за 20 минут вместо 2 недель и 15 000 ₽» (для профи).

**Целевые сегменты:** частник ИЖС/дача (beachhead), проектировщик/архитектор, риелтор/девелопер.

**Единая джоба:** превратить неопределённость «как встанет солнце/тень» в уверенное, обоснованное и задокументированное решение до необратимых затрат.

## 2. Архитектура и структура репозитория

Репозиторий Git на уровне `Sites/` (ветка `main`). Актуальная рабочая версия проекта — в подпапке `insol/`.

Две реализации фронтенда:

- **Ванильная версия** — `insol/index.html`, весь код в одном файле, Three.js с CDN, статический сайт без сборки. Плюс лендинг `landing.html`, статьи/блог (`article-insolyaciya.html`, `blog.html`), портфолио-лендинг.
- **React + Radix порт** — `insol/react-app/` (Vite + React 18 + @radix-ui/themes + three@0.160 + maplibre-gl). Деплой в `/app/`. Это основное направление разработки.

Ключевые файлы React-порта:

- `src/App.jsx` — интерфейс, панели, тарифы, гейтинг Pro.
- `src/engine/astronomy.js` — движок: астрономия (Meeus/SunCalc) + гео-хелперы.
- `src/three/Viewport.jsx` — 3D-сцена, тени, тепловизорная аналитика поверхностей.
- `src/three/SunPath.jsx` — диаграмма пути солнца (небесный купол).
- `src/three/HeatMap.jsx` — годовая тепловая карта.
- `src/three/PlanEditor.jsx` — ручное рисование плана участка/зданий.
- `src/three/ZoneMap.jsx` — рекомендации по зонированию.
- `src/three/MapView.jsx` — реальная карта (MapLibre + OpenFreeMap/MapTiler).
- `cadastre-proxy/worker.js` — Cloudflare Worker-прокси для кадастра (НСПД).
- `admin.html` + `src/admin.jsx` + `src/AdminApp.jsx` — админ-панель `/app/admin` (React + Radix, демо): разделы «Проекты участков» и «Контент и блог».

Документы проекта: `README.md`, `react-port.md`, `jtbd-insolation-service.md`, `gtm-roadmap.md`, `suntrace3d-vs-shadowmap-analysis.md`, `design-guidelines-photoncycle.md`, `Отчет_инсоляция_образец.docx`.

## 3. Функции

**Бесплатно (крючок доверия, без регистрации):** 1 участок по координатам угловых точек; путь и положение солнца в реальном времени; 3D-тени; здания (типовые пресеты, рисование прямоугольником/контуром, двускатные крыши, гизмо перемещения/поворота/масштаба, drag); забор по периметру; инсоляция и длина тени в точке (центроид); контрольные точки; линии отступов 1/3 м; калькулятор экономии; светлая/тёмная/системная тема.

**Pro / платно:** инсоляция по сетке точек и по окнам (фасады); годовая тепловая карта; 3D-аналитика поверхностей («тепловизор»); рекомендации по зонированию (юг — огород, центр — газон, север — посадки); нормативный отчёт (PDF); сохранение/открытие проекта; кадастровая автоподгрузка (по подписке).

**Нормативная база отступов:** дом ≥3 м, прочее ≥1 м (СП 53.13330 / СП 42.13330); ограничение размещения в границах участка; памятка отступов. Астрономия — алгоритм Meeus/SunCalc. Оговорка: расчёты модельные, не заменяют аккредитованную экспертизу.

## 4. Движок (astronomy.js)

Астрономия: положение/движение солнца, восход/закат/полдень, длина дня.

Гео-хелперы (проверены `node --check`): `polyAreaSigned`, `polyArea`, `nearestOnSeg`, `clampToPoly`, `offsetInward`, `plotBasis`, `rectFromDrag`, `pointInPoly`.

Инсоляция: `facadeWindows` + `windowInsolation` + `windowsReport`. Тепловизор: `thermalColor` / `gridSurface` / `roofSlopes3D` (raycast против мешей сцены, учёт рассеянного света).

## 5. Карты и подложки

- 2D-план: подложка-чертёж (JPG/PNG, ширина в метрах, прозрачность).
- Реальная карта: MapLibre + OpenFreeMap (без ключа), спутник/схема через MapTiler (растровые тайлы, free-тариф).
- Ключ MapTiler встроен из сборки (`VITE_MAPTILER_KEY`), поле ввода ключа — как fallback.
- Карта-схема в 3D включена по умолчанию; реальная карта используется как текстура земли в 3D.
- Кадастр: серверный прокси (Cloudflare Worker) к НСПД — по подписке (Pro). Автоподгрузка контура по кадастровому номеру пока в разработке.

## 7. Стратегия (GTM / JTBD)

Beachhead — частник ИЖС/дача. Продаём не «моделирование солнца», а **снятие риска необратимой ошибки** и задокументированное решение.

Методология запуска (синтез каналов Табунова):

1. **DoD вместо описания фич** — конкретный результат за конкретное время, пользователь сам считает ROI.
2. **Правило 180 секунд** — от клика до «вау» ≤3 мин, без регистрации и форм на входе.
3. **Трафик — главный скилл**, старт с поиска/SEO (платёжеспособный спрос), потом перформанс; бюджет старта $100–200.
4. **Отсекать зевак** одним жёстким заголовком.
5. **Fail-fast** — первые юзеры за час, оплаты за день, канал без оплат 2 недели → отрубаем.
6. **Разовая боль → разовый платёж как основа** (не подписка): «Отчёт-паспорт участка» PDF/PNG/GIF, ориентир **490–990 ₽**. Подписка вторична (профи/риелтор, где боль повторяется).
7. Бенчмарки ретеншена (если подписка): день-30 20%+ — золото; подписки 75%+ — ценность.
8. Экономика решает: ориентир CAC ~490 ₽, окупаемость ~x3, цикл покупки ~5 минут.

**Тарифы:** Free (крючок) · Pro (частник/дачник: сетка, тепловая карта, панели, экспорт, сохранение) · Business (профи/риелтор: PDF/DOCX по СанПиН, окна, влияние на соседей, кадастр, white-label, API).

**За что платят охотно (ранжир):** нормативный отчёт по СанПиН → инсоляция по окнам → кадастровая автоподгрузка → тепловая карта + 3D-аналитика + зоны → ограничение по нормам + памятка → влияние на соседей → экспорт/презентабельность → солнечные панели/выработка.

**Конкуренты:** РФ — СИТИС:Солярис, web Insol/Insolaris/Revit, онлайн-калькуляторы; мир — Shadowmap (зрелый лидер, $1.1M seed, 500K+ польз/мес), SunTrace3D (инди, фокус на панелях/ROI, данные PVGIS ЕК), Google Project Sunroof. **Незанятая ниша Инсоляра:** живой 3D-участок + нормативный документ РФ + кадастр/отступы одновременно.
